╠═══════════════════════════════════════════════════════════════════════════╣
║                                                                           ║

║  3. Lambda Invocation  →  FastAPI Application                            ║
║  4. Business Logic  →  Database Query (RDS)                              ║
║  5. Response  ←  Lambda  ←  API Gateway  ←  CloudFront  ←  User          ║
║                                                                           ║
║  Telegram Flow:                                                           ║
║  1. User Message  →  Telegram  →  Webhook  →  API Gateway  →  Lambda     ║
║  2. Lambda  →  AI Agent (GPT-4)  →  Function Calling                     ║
║  3. Function  →  Database Operation  →  Response                         ║
║  4. Response  →  Telegram API  →  User Message                           ║
